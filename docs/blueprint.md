# **App Name**: CodingHawks

## Core Features:

- Hero Section Animation: Create a captivating hero section with animated CTA buttons and background gradients.
- Fields/Teams Showcase: Develop animated cards with hover effects to showcase different fields/teams such as Web Development, AI/ML, and Cybersecurity.
- Dynamic Announcements: Implement a dynamic announcement system using Firebase Firestore to display upcoming events and updates.
- Firestore Integration: Set up Firebase Firestore collections to manage and store announcement data dynamically.
- Scroll Animations: Incorporate scroll-based animations like fade-in and slide-up effects using Framer Motion for a smooth user experience.
- Form Submission Integration: Connect the 'Join Us' section to a Google Form for easy collection of new member applications.

## Style Guidelines:

- Primary color: Deep Purple (#5F1976) for sophistication and depth.
- Accent color: Vibrant Magenta (#A83C8C) to highlight interactive elements.
- Background color: Off-White (#F9F9F9) for a clean and modern look.
- Font: 'Inter', a sans-serif font, for both headers and body text, creating a clean and modern look.
- Note: currently only Google Fonts are supported.
- Use modern and relevant icons for departments and social media links.
- Implement subtle animations using Framer Motion for a smooth user experience.
